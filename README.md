# Mobile IPDV

Mobile construído com **Flutter**.

---

## Tecnologias

- Flutter
- Hive
- GetX

---

### Clonar o repositório

```bash
git clone https://github.com/matheusoreis/ipdv_mobile
cd ipdv-backend
```

### Instalar as dependências

```bash
flutter pub get
```

### Rodar o aplicativo em desenvolvimento

```bash
flutter run
```
